|                                                           |
|                PERSONAJES CON CARACTERÍSTICAS             |
|___________________________________________________________|
|                                                           |
| Este ejercicio consiste en crear y manipular objetos      |
| en PHP que representen personajes con características     |
| específicas, y luego realizar operaciones sobre estos     |
| objetos.                                                  |
|___________________________________________________________|
|                                                           |
| **Requisitos del Ejercicio**                              |
|___________________________________________________________|
|                                                           |
| - Nombre (string)                                         |
| - Edad (int)                                              |
| - Profesión (string)                                      |
| - Habilidad (string)                                      |
|___________________________________________________________|
|                                                           |
| **Operaciones requeridas**:                               |
|___________________________________________________________|
|                                                           |
| - Identificar el personaje más joven                      |
|                                                           |
| - Determinar qué personajes comparten una misma           |
|  habilidad                                                |
