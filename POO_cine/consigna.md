Se trata de presentar un catalogo de peliculas de una cadena de cine.Cada cine tiene un nombre, una poblacion y una lista de peliculas, cada pelicula tiene nombre, duración y director/a. 

El programa debe permitir registrar: 
-> Para cada pelicula, mostrar los datos de las peliculas
-> Para cada cine, proyectar la pelicula más larga
-> Implementar una funcion o metodo que busque el nombre del director en las peliculas de    diferentes cines.

class pelicula:
->nombre
->duracion
->director

class cine:
->nombre
->poblacion
->array de peliculas
